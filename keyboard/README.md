## Hyper keybinds

When pressing Hyper, the following shortcuts become available.

![](hyper_overlay.png)

| Color   | Modifiers                                         |
|---------|---------------------------------------------------|
| Red     | <kbd>Hyper</kbd>                                  |
| Yellow  | <kbd>Hyper</kbd> <kbd>Shift</kbd>                 |
| Blue    | <kbd>Hyper</kbd> <kbd>Ctrl</kbd>                  |
