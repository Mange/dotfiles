## Super keybinds

When holding Super ("Windows key", "Logo key", etc.), the following shortcuts
become available:

![](super_overlay.png)

| Color   | Modifiers                                         |
|---------|---------------------------------------------------|
| Red     | <kbd>Super</kbd>                                  |
| Yellow  | <kbd>Super</kbd> <kbd>Shift</kbd>                 |
| Blue    | <kbd>Super</kbd> <kbd>Ctrl</kbd>                  |
